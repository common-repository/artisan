Ultimate Collection of Essential Blocks for Gutenberg Editor.

== Description ==

Artisan is the progress bar for the Gutenberg editor.

= Available Blocks =

 * **Progress Bar** - Disaply the Progress Bar.
 * **Pricing Table** - Disaply Pricing Table.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.

== Frequently Asked Questions ==

= What are the requirements to use Artisan Blocks? =

You only need to have the latest version of WordPress on your website.
